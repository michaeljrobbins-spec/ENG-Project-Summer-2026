# Macbeth Soliloquy Analysis Workshop

An interactive reading and analysis tool for LIT1000 students at Valencia College. Built as a static web app with no dependencies — just HTML, CSS, and JavaScript.

## What It Does

Students select one of four Macbeth soliloquies and work through a guided analysis workflow:

- **Reading Guide** — Each soliloquy is broken into chunks with plain-language paraphrases, annotations, and clickable clarification buttons that explain unfamiliar words, phrases, and imagery.
- **Three-Lens Analysis** — Students explore the passage through three analytical lenses (Personal, Discursive, and Global), responding to guided questions for each.
- **Writing Scaffold** — After the analysis, students write three paragraphs (one per lens) using an evidence sandwich framework. The tool evaluates their writing against a rubric covering lens coverage, textual evidence, and readability, with feedback for revision.
